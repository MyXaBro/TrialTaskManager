# Менеджер Задач - Триал версия

Учебный проект по дисциплине "Информационная безопасность".

## Описание

Приложение представляет собой менеджер задач с графическим интерфейсом, ограниченный 4 запусками (триал-версия).

### Функционал

- Добавление задач с названием, описанием и приоритетом
- Отметка задач как выполненных
- Удаление задач
- Сохранение задач между сессиями
- Отображение статистики

## Требования

- Go 1.21 или выше
- Windows 10/11
- GCC компилятор (для Fyne) - можно установить через MSYS2 или TDM-GCC

## Установка и запуск

### 1. Установка Go

Скачайте и установите Go с официального сайта: https://go.dev/dl/

### 2. Установка GCC (для компиляции Fyne)

Для Windows рекомендуется TDM-GCC:
- Скачайте с https://jmeubank.github.io/tdm-gcc/
- Установите и добавьте в PATH

### 3. Сборка проекта

```bash
cd TrialTaskManager
go mod tidy
go build -o TaskManager.exe
```

### 4. Запуск

```bash
./TaskManager.exe
```

## Методы защиты

### 1. Хранение счётчика

Счётчик запусков хранится в двух скрытых файлах:
- `%APPDATA%\Microsoft\Windows\Themes\.cache_tm7x`
- `%LOCALAPPDATA%\Temp\~df847291.tmp`

Также создаётся отвлекающий файл:
- `%USERPROFILE%\.config\taskmanager\settings.dat`

### 2. Шифрование

Используется оригинальный алгоритм шифрования:
1. XOR с ключом машины (на основе имени компьютера)
2. XOR с ключом пути файла
3. Побитовая ротация
4. Инверсия каждого 3-го байта
5. Добавление случайного мусора

### 3. Сохранение атрибутов файлов

При записи в файл сохраняются и восстанавливаются:
- Дата создания
- Дата модификации
- Дата последнего доступа

### 4. Защита от анализа

- Обфусцированные имена файлов, функций и переменных
- Отложенные проверки (чтение и проверка в разных местах)
- Отвлекающие операции (ложные чтения реестра, фиктивные файлы)
- Команды goto для усложнения анализа
- Случайные задержки
- Распределённая проверка в разных частях кода

## Тестирование триал-защиты

Для сброса счётчика запусков удалите файлы:
```
%APPDATA%\Microsoft\Windows\Themes\.cache_tm7x
%LOCALAPPDATA%\Temp\~df847291.tmp
```
